## Auto_clicker_HH.ru

Привет! Это Auto_clicker_HH.ru, репозиторий, созданный потому, что мне было скучно и я устал кликать вакансии. Я решил автоматизировать этот процесс, чтобы сделать жизнь немного проще. Наслаждайтесь и пользуйтесь!

Этот скрипт автоматизирует процесс подачи заявок на вакансии на веб-сайте поиска работы hh.ru. Он нажимает на кнопку "Откликнуться", прикрепляет сопроводительное письмо и отправляет его, все это делает автоматически и без лишних усилий с вашей стороны.

#### Использование

- Установите расширение Google Chrome "Javascript Injector", позволяющее вставлять и запускать пользовательский JavaScript на веб-страницах;
- Откройте страницу с вакансиями, на которую вы хотите подать заявление;
- Откройте Javascript Injector, вставьте скрипт и запустите его;
- Наблюдайте, как скрипт автоматически подает заявления на вакансии.
  
Примечание: Вам потребуется заменить строку 'Тут вставляем сопроводительно письмо' на текст вашего сопроводительного письма.
